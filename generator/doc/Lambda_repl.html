Module Lambda_repl (.ml)

module Lambda_repl: sig .. end
Lambda calculus reduction rules applied to formal functions.
See also (in french) for details on formal functions.
val is_simple_value : Syntax.expression -> bool
Recursively defines the expressions to be reduced by the lambda calculus rules.

A function f is said to be a "formal function" if is_simple_value f. Only formal functions are reduced by Lambda_repl.replace and Lambda_repl.normal_order_reduct.

val free_vars : Syntax.expression -> Helper.StringSet.t
Creates a set of all free variables of a formal function.
val substitution : Syntax.expression ->
       Syntax.expression -> Helper.StringSet.elt -> Syntax.expression
Substitutes a variable x for an expression arg in expr using lambda calculus rules.
val replace : (Syntax.expression option * 'a) Syntax.env ->
       Syntax.expression -> Syntax.expression
Replaces all the free variables satisfying Lambda_repl.is_simple_value from an expression by their corresponding expression in env. Uses the Lambda_repl.substitution function to make valid replacements.
val normal_order_reduct : Syntax.expression -> Syntax.expression
Beta-reduces an expression.
